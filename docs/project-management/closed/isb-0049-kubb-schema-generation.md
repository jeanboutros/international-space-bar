## Description

Run `pnpm generate:schemas` using the config from isb-0048 to produce the full set of Kubb-generated Zod schema files. Commit all generated files to source control. Verify `pnpm check` exits 0 with the generated files present (confirming the linter excludes from isb-0048 work in practice).

Generated files are **committed to source control** (not gitignored) — this mirrors the openresponses upstream repository, preserves a full audit trail in PRs, and eliminates any pre-build generation step in CI.

## Acceptance Criteria

- [ ] `pnpm generate:schemas` runs to completion without errors
- [ ] 80 or more schema files are produced under `src/international-space-bar-server/openresponses/generated/zod/`
- [ ] All generated files are staged and committed (`git add src/international-space-bar-server/openresponses/generated/`)
- [ ] `.gitignore` does **not** list `src/international-space-bar-server/openresponses/generated/` (generated files must be tracked)
- [ ] `pnpm check` exits 0 with all generated files present in the working tree

## Files Affected

- `src/international-space-bar-server/openresponses/generated/` (new directory — all contents generated by Kubb)
