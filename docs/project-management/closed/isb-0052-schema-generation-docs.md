## Description

Document the Kubb schema generation toolchain: how to regenerate schemas, where the output lives, how the ISB wrapper overrides work, and what to do when the OpenAPI spec changes. Update the project-wide reference docs (`docs/technical-stack.md`, `AGENTS.md`) to reflect the new toolchain and the convention that generated files are excluded from linters.

**Docs standards flag resolution:** `AGENTS.md` is the correct home for code conventions (including the generated-files linter-exclude convention). No `docs/standards/` directory is created as part of this ticket set.
